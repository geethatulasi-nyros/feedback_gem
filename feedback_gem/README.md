## Installation
Add this line to your application's Gemfile:

```ruby
gem 'feedback_gem'
```

And then execute:
```bash
$ bundle
``

Or install it yourself as:
```bash
$ gem install feedback_gem
```
## Migrations
$ rails feedback_gem:install:migrations

$ rails db:migrate

## Routes
mount FeedbackGem::Engine => "/feedback_gem"
